## 题目描述

<p><code>Numbers</code> 表保存数字的值及其频率。</p>

<pre>+----------+-------------+
|  Number  |  Frequency  |
+----------+-------------|
|  0       |  7          |
|  1       |  1          |
|  2       |  3          |
|  3       |  1          |
+----------+-------------+
</pre>

<p>在此表中，数字为 <code>0, 0, 0, 0, 0, 0, 0, 1, 2, 2, 2, 3</code>，所以中位数是 <code>(0 + 0) / 2 = 0</code>。</p>

<pre>+--------+
| median |
+--------|
| 0.0000 |
+--------+
</pre>

<p>请编写一个查询来查找所有数字的中位数并将结果命名为 <code>median</code> 。</p>
